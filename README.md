
### 介绍 

 Vue3.4、TypeScript、Vite5、Pinia、Element-Plus 常用组件、Hooks、指令、动态路由、按钮级别权限控制等功能。


### 在线预览 

- Link：https://admin.spicyboy.cn



- **Install：**

```text
pnpm install
```

- **Run：**

```text
pnpm dev
pnpm serve
```

- **Build：**

```text
# 开发环境
pnpm build:dev

# 测试环境
pnpm build:test

# 生产环境
pnpm build:pro
```

- **Lint：**

```text
# eslint 检测代码
pnpm lint:eslint

# prettier 格式化代码
pnpm lint:prettier

# stylelint 格式化样式
pnpm lint:stylelint
```
